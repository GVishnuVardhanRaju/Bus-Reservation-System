The Bus Reservation System is a simple menu-driven application developed in the C programming language that simulates a basic seat reservation process for a bus.

The program allows users to view available seats and reserve a seat through a command-line interface. It demonstrates the use of arrays, functions, loops, and conditional statements in C programming.

🌟 Features

📋 Display Available Seats
Shows all bus seats and their current status.

🎟 Reserve a Seat
Allows users to select and reserve a specific seat.

⚠ Seat Availability Check
Prevents double booking of already reserved seats.

🔢 Seat Limit
Supports up to 50 seats.

🖥 Menu Driven Interface
Simple interactive menu for easy navigation.

⚙️ How the Program Works

The system initializes 50 seats as available.

Users are presented with a menu:

Display available seats

Reserve a seat

Exit the program

When a user selects a seat:

The system checks if it is available.

If available, the seat is reserved.

If already reserved, the system shows an error message.
